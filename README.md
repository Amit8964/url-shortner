# url-shortner

A URL- shortener created using 
php and synced with Mysql Database.

# Why use it?

Great tool for shortening a very long url
to just 5 characters.

# Features 

* Shortern a long URL using the web app.

* Gets saved in Mysql Database for 
quick access.

# Built With

* php 
* Mysql Database
* javascript
